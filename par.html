VIP Labor + Cleaning Parser v1.23 revisions

  • v1.23: Tables are editable before copy/export. Subdivision cells use exact values from the loaded Subdivisions sheet, task cells use exact values from Task Types / cleaning task choices, and Notes/AM/Hours/Lot can be adjusted directly.
  • v1.23: Fixed parenthetical cleaning notes like Miller remove-floor-covering notes, fixed AM text being pulled into subdivision names, preserved full Other notes like porch/drive wording, and kept exact loaded subdivision/task spelling.
  • Moved optional uploads, server paths, Alias Manager, and Clear Saved Files into the gear settings panel.
  • Removed the large green loaded popup and replaced it with a compact loaded-status line.
  • Custom powerwash descriptions that do not cleanly match approved task types now become Other with the raw task in Notes.
  • Addresses in raw lines are preserved in Notes for warranty/lived-in jobs.
  • Scrap Subdiv lines now split correctly: Miller is the subdivision, scrap subdiv goes to Notes, Lot is 0, and Task is Other.
  • PO/invoice/price junk, including PO#?????? and unknown PO placeholders, is stripped from Notes more aggressively.
  • Hours stay blank for every row.
  • v1.11: Fixed condensed lines where the task appears before the dash, like 371 Glacier**MI p/w int- note. The parser now moves p/w int out of the subdivision and classifies it as the task.
  • v1.12: Added condensed MI splitting, PM handling, and alias exact lookups that prefer the loaded subdivision row.
  • v1.13: Rebuilt the condensed MI splitter so slash-compressed lines expand reliably. Bare p/w now becomes Other with a clarification note instead of guessing Powerwash Garage.
  • v1.17: Condensed same-lot double tasks now split correctly, like /392-scrap/p/w int becoming two rows for lot 392: Scrap and Power Wash Interior.
  • v1.18: Same-lot condensed task bundles now split, like 22 Pinnacle-p/w int/car/scrap becoming three rows: Power Wash Interior, Carpet Prep, and Scrap.
  • v1.20: Added a separate Cleaning tab with four input boxes: Originals, Recleans, Windows, and Other. Cleaning exports to the Cleaning Jobs sheet layout separately from Labor.
  • v1.19: Restored builder-specific bare p/w rules for Pulte/DR/Ryan/Rockford/Epcon, kept bare MI p/w as review, added JV Applewood alias, and split same-lot known task leftovers like car relay into separate rows.
  • v1.17: Scrap maps to Scrap- inside house/floor coverings/garage and Crawl maps to Cover Crawl, using the task sheet names instead of outputting plain shorthand.
  • v1.17: Ridge at Glacier is hard-locked to Glacier Pointe *MI *Plain City so it cannot fall back to Schottenstein.
