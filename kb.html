Installation

  1. Go to the VPDesk download page (visual-planning.com/support-portal/vpdesk-new-launcher) and click Download.
  2. Double-click the installer and follow on-screen prompts until the launcher appears.
  3. Click the three-dot menu (⋮) in the launcher’s top-right.
  4. Select Add URL, enter https://us2.visual-planning.com/provantage/index.jsp.
  5. Back in the launcher, click Planner and sign in with your credentials.

Usage Features

Add a Job to Tomorrow’s Schedule
  1. Select the department (Cleaning, Labor, Windows & Commercial) from the left sidebar.
  2. Ensure the date at top is tomorrow; if not, click Planner and press → to advance.
  3. In the LOTS toolbar, right-click, open Search → Lot.
  4. Enter the lot number in search.
  5. Drag the desired task from the task pane onto the lot entry.
Add a Lot to Planner if Not Loaded
  1. If the lot isn’t found in search, right-click anywhere in the left planner pane.
  2. Hover over the + icon and click Lot.
  3. Fill out all fields: Floorplan, Builder, Subdivision, City, Zip, Address.
  4. Click OK to save. The new lot will update and appear for scheduling.
Change a Task Type
  1. Click the task you want to modify in the schedule.
  2. In the Task Details pane, click on the current task name.
  3. Select the new task type (e.g., Original Dry Clean) from the dropdown menu.
  4. Click OK to save the change.
Edit Crew Sizes & Members
  1. Navigate to Crew Schedule via the top nav bar for your department.
  2. Click the crew you want to update.
  3. In the right panel, adjust:
    • Active Tomorrow: toggle on/off.
    • Total Man Hours: enter available hours.
    • Crew Size: set number of members.
  4. Under Crew Members, add or remove team members. If a name isn’t listed, contact Noah or Becky to add them.
  5. Click OK to save changes.
AutoScheduling
  1. After updating all crews and tomorrow’s schedule, navigate to Crew Schedule from the top nav bar.
  2. In the list of TBD jobs, right-click any job entry.
  3. Hover over Exchange → Export events → Automatic Scheduling.
  4. The automatic scheduling process will run for about one minute. Wait until it completes before making further changes.
Reset Schedule
  1. Go to Crew Schedule.
  2. Right-click a TBD job.
  3. Exchange → Export events → Reset Schedule.
  4. This clears assignments so you can adjust off-days/hours before rerunning.
Checking a Lot’s Timeline
  1. On Cleaning or Labor schedule, search for the lot.
  2. Double-click the lot.
  3. Select Lot Timeline.
  4. Switch to Cleaning Record.
Adding a Model to the Planner
  1. Go to your department’s schedule (Cleaning, Labor, etc.).
  2. Right-click the LOTS bar → Search → Letter, then type Model.
  3. If Model doesn’t appear, right-click the planner pane → hover + → click Lot.
  4. In the Add Lot form, enter 0 for Lot Number and Model for Letter; fill other fields as needed and click OK.

VPGO (Field/Phone Version)

Installation

  1. Download and install VPGO U from the App Store or Google Play Store.
  2. Open VPGO U and tap the + icon.
  3. Select Flash QR Code.
  4. Scan QR Code below.
  5. Sign in with your VPGO credentials.

Usage

  1. Sign In: Enter your VPDesk username and password, then tap Sign In.
  2. View Jobs: Tap the Jobs tab to see all assignments for the day.
  3. Select a Job: Tap the job to view details and workflow.
  4. Navigate: Tap the map icon to open the job location in your maps app.
  5. Check In: Tap Check In, validate your location, then tap the checkmark.
  6. Check Out: After completing work, tap Check Out, validate, and save.
