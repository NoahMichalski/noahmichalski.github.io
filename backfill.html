Calendar → Floorplans Backfill Checker

Sign into Google Calendar, pull old job events, upload your Floorplans sheet, then find lots that were completed before the sheet existed. Pulte and DR Horton get special address-based floorplan matching.

1. Google setup

For Google sign-in, host this file from http://localhost or your website and add that origin to your Google OAuth client. Double-clicking the file may not work because Google blocks most file:// origins.

Scope used: https://www.googleapis.com/auth/calendar.readonly. This tool only reads events and keeps the comparison in your browser.

Date range

This keeps recleans/windows from flooding Needs Review. Raw Events still shows everything pulled.

2. Calendars

Sign in first, then load calendars.

3. Floorplans sheet

Expected columns can be named normally: Builder, Subdivision, Lot, Floorplan, Original, Reclean, Windows, Address. The tool does fuzzy header matching.

Aliases / cleanup rules

Expected columns: Builder, Subdivision, Shortened. This adds alias lines like Shortened = exact Floorplans subdivision so older calendar titles match your current sheet names.
